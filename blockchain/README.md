# Stellar SMEs - Blockchain Layer

Utilities and scripts for interacting with the Stellar network.

## Scripts

### Fund Testnet Account
```bash
npm run fund-account <PUBLIC_KEY>
```

### Check Account Balance
```bash
npm run check-balance <PUBLIC_KEY>
```
